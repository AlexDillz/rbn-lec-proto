## О проекте

**RUBIN QR LEC** — это сайт, через который участники быстро попадают к нужным лекциям, материалам и дополнительным разделам ШВМ.

Сценарий простой:

1. Участник сканирует QR-код.
2. Открывает главную страницу со списком лекций.
3. Переходит в нужную лекцию, карточки лекторов, контакты или экзаменационные билеты.

Проект специально сделан **максимально простым в поддержке**: без сборщиков, без backend, без сложной инфраструктуры.

- главная страница со списком лекций

## Структура проекта

```text

└── assets/
    ├── lecs/
    ├── personas/
    ├── cards/
    └── ...


Как это работает
index.html
Главная точка входа. Загружает список лекций из lectures.json и собирает карточки на странице.

lectures.json
Конфиг со списком лекций:

название
краткое описание
ссылка на страницу
статус доступности
lectures/*.html
Отдельные страницы лекций с единым стилем, навигацией и дополнительными материалами.

style.css
Общий стиль сайта: главная страница, карточки, лекции, разделы, адаптивность.

theme.js
Переключение светлой и тёмной темы с сохранением выбора пользователя.

protect-local.js
Локальная защита и логика для ограниченного просмотра отдельных страниц.

Локальный запуск
python3 -m http.server 8000
После запуска сайт будет доступен по адресу:

http://localhost:8000
Как добавить новую лекцию
Создать новую страницу в папке lectures/
Добавить запись в lectures.json
При необходимости положить материалы в assets/lecs/
Проверить отображение карточки на главной странице
Проверить переход из списка лекций
